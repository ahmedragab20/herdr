# herdr

Dotfiles for the [herdr](https://herdr.dev/) terminal
multiplexer: keybindings, theme, prefix actions, and the pinned plugin source.

## Key bindings (cheat sheet)

Prefix: <kbd>Cmd</kbd>+<kbd>E</kbd>

- <kbd>prefix</kbd>+<kbd>f</kbd> — open the file viewer in a split
- <kbd>prefix</kbd>+<kbd>shift</kbd>+<kbd>f</kbd> — open the file viewer in its own tab
- <kbd>prefix</kbd>+<kbd>c</kbd> / <kbd>x</kbd> — new / close tab
- <kbd>prefix</kbd>+<kbd>v</kbd> / <kbd>-</kbd> — split vertical / horizontal
- <kbd>prefix</kbd>+<kbd>w</kbd> / <kbd>n</kbd> / <kbd>d</kbd> — workspace picker / new / close
- <kbd>prefix</kbd>+<kbd>s</kbd> — settings, <kbd>prefix</kbd>+<kbd>shift</kbd>+<kbd>r</kbd> — reload config

See [`config.toml`](config.toml) for the full mapping.
